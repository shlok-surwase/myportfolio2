# Shlok Praveen Surwase — Portfolio

Personal portfolio website built with HTML, CSS & JavaScript.

## 🌐 Deploy on GitHub Pages
1. Upload all files to a GitHub repo named `portfolio`
2. Settings → Pages → Branch: main → Save
3. Live at: `https://shlok-surwase.github.io/portfolio`

## 📁 Structure
```
portfolio/
├── index.html
├── style.css
├── script.js
├── README.md
└── assets/
    ├── profile.jpg
    ├── profile2.png
    ├── project-roseai.png
    ├── project-velocity.jpg
    ├── cert-cisco.jpg
    ├── cert-coursera.jpg
    └── cert-disha.jpg
```

## ✨ Features
- Editorial serif typography (Cormorant Garamond)
- Magnetic hover effect on hero photo
- Menu overlay with image changing per section
- Scroll-triggered reveal animations
- Certificate lightbox viewer
- Copy email button
- Fully responsive
- Custom cursor
